Manual rápido

El aplicativo inicia con datos demo validados contra las fuentes anexas. También puede cargar los libros Excel institucionales para recalcular el dashboard en tiempo real.

  1. Cargue uno o varios archivos Excel mediante el botón superior o arrastrando al panel lateral.
  2. Revise el módulo de conciliación para identificar diferencias entre ACRESC, Resultados y Formato 514.
  3. Use filtros globales para enfocar cliente, línea, calificación, sector o mora.
  4. Exporte el dashboard a PDF o las tablas a Excel.
